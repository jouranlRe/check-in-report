# 期刊签收统计报告

## 文件说明

| 文件 | 说明 |
|---|---|
| `index.html` | 期刊签收统计报告页面，双击即可在浏览器打开 |
| `styles.css` | 页面样式 |
| `scripts.js` | 页面交互脚本 |

## 概览卡片

页面顶部 10 张卡片展示核心统计数据：期刊总种数、中文/外文期刊数、到刊总期数、疑似漏发期数、本月签收情况、整体完成率、超 60 天未签收期刊数、零签收期刊数。

## 到刊趋势图表

两幅折线图展示 2026 年每月到刊的种数和期数变化趋势，中文和外文分色显示。

## 异常台账

四个标签页：
- **漏刊预警** — 列出疑似漏发的期刊及漏发期次详情，可按刊名、漏发数量排序
- **停刊汇总** — 列出已停刊的期刊及停刊期数，可按刊名排序
- **合刊统计** — 列出存在合刊情况的期刊及合刊说明，可按刊名排序
- **超 60 天无签收** — 列出长期未签收的期刊及最后到刊日期，可按刊名、最后到刊时间排序

## 期数网格

中文和外文分 tab 展示，每行一本期刊，1–12 月用彩色方块表示签收状态：绿色=已签收、黄色=合刊、红色=停刊、灰色=未到。

- **刊名固定** — 横向滚动时刊名列固定在左侧不动，方便对照
- **刊名搜索** — 输入刊名实时过滤，匹配行高亮显示
- **拖拽滚动** — 按住方块区域左右拖拽横向查看
- **点击查看** — 点击任意方块弹出详情窗，显示刊名、期号、签收状态和对应的到刊时间

## 到刊明细

中文/外文明细表格，列出每种期刊的邮发号/刊号、刊名、完成率（已到/应到）、是否停刊、是否漏发。可按刊名、完成率排序。

## 签收时间线

按日期倒序展示所有签收记录，每天汇总展示当日的签收期刊及期次。

- **刊名搜索** — 输入刊名实时过滤
- **月份筛选** — 按下拉菜单选择月份过滤
- **动态统计** — 右上角实时显示当前筛选结果的总种数和总期数

## 页面导航

页面左侧（手机端底部）设有悬浮导航栏，包含概览、趋势、异常、网格、明细、时间线六个快捷入口，滚动时自动高亮当前所在区块。

标签页切换时 URL 自动记录状态，刷新页面后自动恢复。
