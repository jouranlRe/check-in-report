# 期刊签收统计报告

## 文件说明

| 文件 | 说明 |
|---|---|
| `index.html` | 期刊签收统计报告主页面，双击即可在浏览器打开 |
| `styles.css` | 页面样式 |
| `scripts.js` | 页面交互脚本 |

## 页面功能

### 概览卡片
页面顶部 10 张卡片展示核心数据：期刊种数、到刊总期数、本月签收情况、完成率、漏发预警、零签收等。

### 到刊趋势图表
两幅折线图分别展示 2026 年各月到刊种数和期数的变化趋势，中文/外文分色显示。

### 异常台账
四个标签页分别展示：
- **漏刊预警** — 疑似漏发期刊及详情，支持按刊名、漏发数量排序
- **停刊汇总** — 已停刊期刊及停刊期数，支持按刊名排序
- **合刊统计** — 合刊期刊及合刊说明，支持按刊名排序
- **超60天无签收** — 长期无签收期刊，支持按刊名、最后到刊时间排序

### 期数网格
按月份显示每本期刊全年期数的签收状态（已签收/合刊/停刊/未到），支持：
- **搜索** — 输入刊名实时过滤
- **拖拽滚动** — 按住空白区域左右拖拽横向滚动
- **点击查看** — 点击任意色块弹出详情（刊名、期号、状态）
- **色块放大** — 方块和数字经过放大优化，便于阅览

### 到刊明细
中文/外文期刊详细签收数据表格，支持按刊名、完成率排序（完成率自动提取百分比数值）。

### 签收时间线
按日期倒序展示所有签收记录，支持：
- **刊名搜索** — 输入刊名实时过滤
- **月份筛选** — 按月份下拉选择过滤
- **动态统计** — 右上角实时显示过滤后的种数和期数
- **日期动态更新** — 筛选后括号内数字自动更新为当前筛选结果的数量

### 导航与交互
- 页面左侧（手机端底部）设有浮动导航栏，点击可快速跳转到各模块
- 滚动页面时导航栏自动高亮当前所在区块
- 切换标签页时 URL 自动记录状态，刷新页面后自动恢复
